潞行丝语——潞绸非遗新文创

潞绸，是山西古潞安府、泽州府地区，今山西长治、晋城地区传统丝织技术的代称，因其精湛的工艺，有”南淞江，北潞安，衣天下”的佳话。明朝前中期，潞绸产业快速发展丝织业得到发展，成为中国北方主要的丝织品重镇。明代科学家徐光启在《农政全书》中写道”东南之机，三吴越闽最火，西北之机潞最工”，”潞城机抒斗巧，织作纯丽，衣被天下”。潞绸与杭缎、蜀锦并称为中国三大名绸，有着”三分天下有其一”的辉煌历史 。2013年，潞绸手工织造技艺被认定为山西省省级非物质文化遗产。次年，该技艺进一步被纳入第四批国家级非物质文化遗产扩展名录。本项目通过深入发掘中华传统纹样，整合美术资源，成功设计出涵盖笔记本、录取通知书、婚书、发饰、香囊等品类丰富的潞绸文创产品。
